# **App Name**: Neon Access

## Core Features:

- Invite Code Generation: Generate unique invite codes with a 'NA' prefix followed by 5 alphanumeric characters.
- Code Status Tracking: Monitor the status of each invite code, which can be 'Used', 'Not Used', or 'Expired'.
- Automated Email Invitations: Send out beta invitation emails that embed the unique invite codes. Use a tool to decide the email content.
- Waitlist User Registration: Collect user details (name, email, company, phone) for the beta waitlist.
- Real-Time Dashboard: Show a real-time dashboard with key metrics: total codes, usage rate, active codes, emails sent.

## Style Guidelines:

- Background color: Very dark gray (#121212) to enhance the neon effect.
- Primary color: Electric Purple (#A729F0) to give the dashboard a futuristic feel.
- Accent color: Cyan (#29F0E3) for interactive elements to pop against the dark background.
- Font: 'Space Grotesk', sans-serif. Use for both headings and body text, to maintain a tech-focused aesthetic.
- Use minimalist vector icons with neon outlines for key actions and statuses.
- Mobile-first responsive layout to ensure accessibility across devices.
- Subtle transitions and loading animations for a smooth user experience.
